# DSL-085: proposer_inclusion_reward Formula

> **Authoritative:** [DSL-085](../NORMATIVE.md#DSL-085) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §8.4, §2.3, §22.9

---

## Summary

Block proposer earns an inclusion bonus for each attestation they include first:
```
proposer_reward = attester_base_reward * PROPOSER_WEIGHT / (WEIGHT_DENOMINATOR - PROPOSER_WEIGHT)
                = base * 8 / 56
```

---

## Specification

```rust
pub fn proposer_inclusion_reward(attester_base_reward: u64) -> u64 {
    attester_base_reward
        .saturating_mul(PROPOSER_WEIGHT)
        .saturating_div(WEIGHT_DENOMINATOR - PROPOSER_WEIGHT)
}
```
