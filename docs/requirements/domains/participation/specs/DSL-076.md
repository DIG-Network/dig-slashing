# DSL-076: classify_timeliness TIMELY_TARGET

> **Authoritative:** [DSL-076](../NORMATIVE.md#DSL-076) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §8.1, §2.5, §22.9

## Summary

Sets `TIMELY_TARGET` iff `delay ∈ [1, 32]` AND `target_is_canonical` is true.

---

## Specification

```rust
if delay >= MIN_ATTESTATION_INCLUSION_DELAY
   && delay <= TIMELY_TARGET_MAX_DELAY_SLOTS
   && target_is_canonical {
    flags.set(TIMELY_TARGET_FLAG_INDEX);
}
```

---

## Acceptance Criteria

- [ ] delay=1 + canonical target → TARGET set.
- [ ] delay=32 + canonical → set (boundary).
- [ ] delay=33 → NOT set.
- [ ] delay=10 + non-canonical target → NOT set.
