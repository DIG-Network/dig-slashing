# DSL-082: compute_flag_deltas Reward on Hit

> **Authoritative:** [DSL-082](../NORMATIVE.md#DSL-082) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §8.3, §2.3, §22.9

---

## Summary

Per validator, per epoch:
```
if SOURCE hit: reward += base * 14 / 64
if TARGET hit: reward += base * 26 / 64
if HEAD   hit: reward += base * 14 / 64
```

All three hit = reward ≈ base * 54 / 64 (the 2 unassigned weight units represent the unused sync-committee slot).

---

## Specification

```rust
for (idx, flags) in previous_epoch_participation.iter().enumerate() {
    let eff_bal = effective_balances.get(idx as u32);
    let base = base_reward(eff_bal, total_active_balance);
    let mut reward: u64 = 0;
    if flags.is_source_timely() { reward += base * TIMELY_SOURCE_WEIGHT / WEIGHT_DENOMINATOR; }
    if flags.is_target_timely() { reward += base * TIMELY_TARGET_WEIGHT / WEIGHT_DENOMINATOR; }
    if flags.is_head_timely()   { reward += base * TIMELY_HEAD_WEIGHT   / WEIGHT_DENOMINATOR; }
    // penalties computed in DSL-083
    out.push(FlagDelta { validator_index: idx as u32, reward, penalty: /* see DSL-083 */ });
}
```

## Acceptance Criteria

- [ ] SOURCE hit alone → `base * 14 / 64`.
- [ ] TARGET hit alone → `base * 26 / 64`.
- [ ] HEAD hit alone → `base * 14 / 64`.
- [ ] All three → `base * 54 / 64`.
- [ ] Integer math (no rounding surprises for test inputs).
