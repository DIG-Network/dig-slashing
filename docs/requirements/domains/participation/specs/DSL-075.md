# DSL-075: classify_timeliness TIMELY_SOURCE

> **Authoritative:** [DSL-075](../NORMATIVE.md#DSL-075) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §8.1, §2.5, §22.9

---

## Summary

`classify_timeliness` sets `TIMELY_SOURCE` flag iff `delay ∈ [MIN_ATTESTATION_INCLUSION_DELAY, TIMELY_SOURCE_MAX_DELAY_SLOTS]` (= `[1, 5]`) AND `source_is_justified` is true. `delay = inclusion_slot - data.slot`.

---

## Specification

```rust
let delay = inclusion_slot.saturating_sub(data.slot);
let mut flags = ParticipationFlags::default();
if delay >= MIN_ATTESTATION_INCLUSION_DELAY
   && delay <= TIMELY_SOURCE_MAX_DELAY_SLOTS
   && source_is_justified {
    flags.set(TIMELY_SOURCE_FLAG_INDEX);
}
```

---

## Acceptance Criteria

- [ ] delay=1 + justified → SOURCE set.
- [ ] delay=5 + justified → SOURCE set (boundary).
- [ ] delay=6 + justified → SOURCE NOT set.
- [ ] delay=3 + unjustified → SOURCE NOT set.
- [ ] delay=0 → SOURCE NOT set (MIN_ATTESTATION_INCLUSION_DELAY).
