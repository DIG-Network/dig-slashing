# DSL-086: WEIGHT_DENOMINATOR = 64 (No Sync Committee)

> **Authoritative:** [DSL-086](../NORMATIVE.md#DSL-086) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §2.3, §22.9

---

## Summary

DIG does NOT ship a sync committee. `WEIGHT_DENOMINATOR = 64` matches Ethereum, but assigned weights sum to only 62 (= 14+26+14+8). The 2 remaining units correspond to the unused `SYNC_REWARD_WEIGHT` slot in Ethereum Altair — explicitly reserved to preserve formula compatibility but never assigned in DIG.

---

## Specification

```rust
pub const TIMELY_SOURCE_WEIGHT: u64 = 14;
pub const TIMELY_TARGET_WEIGHT: u64 = 26;
pub const TIMELY_HEAD_WEIGHT:   u64 = 14;
pub const PROPOSER_WEIGHT:      u64 = 8;
pub const WEIGHT_DENOMINATOR:   u64 = 64;
// Unassigned: 2 units reserved for Ethereum's SYNC_REWARD_WEIGHT — not used in DIG.
```
