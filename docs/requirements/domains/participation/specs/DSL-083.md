# DSL-083: compute_flag_deltas Penalty on Miss (Head Exempt)

> **Authoritative:** [DSL-083](../NORMATIVE.md#DSL-083) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §8.3, §22.9

---

## Summary

Penalty rules (Ethereum Altair parity):
- SOURCE miss → `penalty += base * 14 / 64`
- TARGET miss → `penalty += base * 26 / 64`
- HEAD miss → **NO PENALTY** (timing is too network-dependent to fairly punish)

---

## Specification

```rust
let mut penalty: u64 = 0;
if !flags.is_source_timely() { penalty += base * TIMELY_SOURCE_WEIGHT / WEIGHT_DENOMINATOR; }
if !flags.is_target_timely() { penalty += base * TIMELY_TARGET_WEIGHT / WEIGHT_DENOMINATOR; }
// HEAD miss: no penalty — Ethereum parity.
```

## Acceptance Criteria

- [ ] SOURCE missed → penalty `base * 14 / 64`.
- [ ] TARGET missed → penalty `base * 26 / 64`.
- [ ] HEAD missed → penalty unchanged (0 from this flag).
- [ ] All missed → penalty `base * 40 / 64`.
