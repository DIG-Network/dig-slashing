# DSL-133: ValidatorEntry::restore_status

> **Authoritative:** [DSL-133](../NORMATIVE.md#DSL-133) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §15.1

---

## Summary

`restore_status()` clears a `Slashed` state back to `Active`. Returns `true` iff the status changed. Calling on already-Active returns `false` (idempotent).

---

## Specification

```rust
pub trait ValidatorEntry {
    fn restore_status(&mut self) -> bool;
}
```

---

## Acceptance Criteria

- [ ] Slashed → Active returns `true`.
- [ ] Active → Active returns `false`.
- [ ] `is_slashed()` returns false after restore_status on previously-slashed entry.
- [ ] Idempotent across repeated calls.
