# DSL-142: RewardClawback::claw_back Partial Return

> **Authoritative:** [DSL-142](../NORMATIVE.md#DSL-142) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §12.2

---

## Summary

`RewardClawback::claw_back(principal_ph, amount)` returns mojos ACTUALLY deducted from the principal's reward account (0..=amount). Partial return is legitimate — the principal may have already withdrawn some or all of the optimistic reward. Adjudicator (DSL-073) absorbs the shortfall from the forfeited reporter bond.

---

## Specification

```rust
pub trait RewardClawback {
    fn claw_back(&mut self, principal_ph: Bytes32, amount: u64) -> u64;
}
```

---

## Acceptance Criteria

- [ ] Full balance → returns `amount`.
- [ ] Partial balance → returns `balance < amount`.
- [ ] Empty balance → returns 0.
- [ ] Saturating subtraction (no underflow).
- [ ] Multiple clawbacks compose (second sees post-first balance).
