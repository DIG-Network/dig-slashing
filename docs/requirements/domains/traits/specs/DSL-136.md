# DSL-136: ValidatorView::get / get_mut

> **Authoritative:** [DSL-136](../NORMATIVE.md#DSL-136) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §15.1

---

## Summary

`ValidatorView::get(idx)` returns `Some(&dyn ValidatorEntry)` for `idx < validator_set.len()`; `None` otherwise. `get_mut` mirrors (mutable borrow). `len()` returns the registered validator count.

---

## Specification

```rust
pub trait ValidatorView {
    fn get(&self, idx: u32) -> Option<&dyn ValidatorEntry>;
    fn get_mut(&mut self, idx: u32) -> Option<&mut dyn ValidatorEntry>;
    fn len(&self) -> usize;
}
```

---

## Acceptance Criteria

- [ ] Live idx → Some.
- [ ] Out-of-range idx → None.
- [ ] `len()` consistent with max valid index.
- [ ] `get_mut` allows mutation observable through subsequent `get`.
