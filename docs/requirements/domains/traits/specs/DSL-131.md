# DSL-131: ValidatorEntry::slash_absolute Saturation

> **Authoritative:** [DSL-131](../NORMATIVE.md#DSL-131) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §15.1

---

## Summary

`slash_absolute(amount_mojos, epoch)` is the canonical stake debit. If `amount_mojos > current_stake`, the debit saturates at `current_stake` (no underflow, no panic). Return value equals mojos actually debited.

---

## Specification

```rust
pub trait ValidatorEntry {
    fn slash_absolute(&mut self, amount_mojos: u64, epoch: u64) -> u64;
}
```

Semantics: `let debited = min(amount_mojos, current_stake); stake -= debited; status = Slashed { epoch }; return debited;`

---

## Acceptance Criteria

- [ ] amount < stake → debit == amount, stake -= amount.
- [ ] amount == stake → debit == stake, stake == 0.
- [ ] amount > stake → debit == stake (saturating).
- [ ] Return value is actual mojos debited.
- [ ] Status transitions to `Slashed { epoch }`.
