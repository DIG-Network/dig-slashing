# DSL-138: PublicKeyLookup::pubkey_of

> **Authoritative:** [DSL-138](../NORMATIVE.md#DSL-138) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §15.2

---

## Summary

`PublicKeyLookup::pubkey_of(idx)` returns the validator's BLS public key. Blanket impl `impl<T: ValidatorView + ?Sized> PublicKeyLookup for T` delegates to `ValidatorEntry::public_key`.

---

## Specification

```rust
pub trait PublicKeyLookup {
    fn pubkey_of(&self, idx: u32) -> Option<&PublicKey>;
}

impl<T: ValidatorView + ?Sized> PublicKeyLookup for T {
    fn pubkey_of(&self, idx: u32) -> Option<&PublicKey> {
        self.get(idx).map(|e| e.public_key())
    }
}
```

---

## Acceptance Criteria
