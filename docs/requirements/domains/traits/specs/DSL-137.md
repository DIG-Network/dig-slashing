# DSL-137: EffectiveBalanceView::get / total_active

> **Authoritative:** [DSL-137](../NORMATIVE.md#DSL-137) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §15.2

---

## Summary

`EffectiveBalanceView::get(idx)` returns the validator's effective balance in mojos (0 for inactive or unknown). `total_active()` returns the sum of all active-validator effective balances. Consumed by base-reward and correlation-penalty formulas.

---

## Specification

```rust
pub trait EffectiveBalanceView {
    fn get(&self, idx: u32) -> u64;
    fn total_active(&self) -> u64;
}
```

---

## Acceptance Criteria

- [ ] `get(live_idx)` returns effective balance.
- [ ] `get(unknown_idx)` returns 0 (no panic).
- [ ] `total_active()` equals sum of active balances.
- [ ] Both methods are read-only (no internal mutation).
