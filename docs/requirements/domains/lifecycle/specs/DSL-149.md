# DSL-149: SlashingManager::is_slashed

> **Authoritative:** [DSL-149](../NORMATIVE.md#DSL-149) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §7.2

---

## Summary

`SlashingManager::is_slashed(idx, validator_set)` delegates to `ValidatorView::get(idx)?.is_slashed()`. Returns `false` for unknown validator indices. Reads — no mutation.

---

## Specification

```rust
pub fn is_slashed(&self, idx: u32, vs: &dyn ValidatorView) -> bool {
    vs.get(idx).map(|e| e.is_slashed()).unwrap_or(false)
}
```

---

## Acceptance Criteria

- [ ] Slashed validator → true.
- [ ] Active validator → false.
- [ ] Unknown index → false (no panic).
- [ ] Read-only — no side effects on self or validator_set.
