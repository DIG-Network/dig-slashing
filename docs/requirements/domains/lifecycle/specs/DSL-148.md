# DSL-148: SlashingManager::new + set_epoch

> **Authoritative:** [DSL-148](../NORMATIVE.md#DSL-148) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §7.2

---

## Summary

`SlashingManager::new(current_epoch)` constructs an empty manager: empty `processed`, empty `PendingSlashBook`, empty `slashed_in_window`, `current_epoch` set to the supplied value. `set_epoch(epoch)` updates `current_epoch`; subsequent `submit_evidence` uses the new value for lookback enforcement (DSL-011).

---

## Specification

```rust
impl SlashingManager {
    pub fn new(current_epoch: u64) -> Self;
    pub fn set_epoch(&mut self, epoch: u64);
}
```

---

## Acceptance Criteria

- [ ] `new(0)` → all internal maps/books empty; `current_epoch == 0`.
- [ ] `new(100)` → same, `current_epoch == 100`.
- [ ] `set_epoch(5)` → `current_epoch == 5`.
- [ ] `set_epoch` is monotonic-agnostic (caller enforces forward progression).
