# DSL-123: BondEscrow::release Full on Finalise

> **Authoritative:** [DSL-123](../NORMATIVE.md#DSL-123) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §12.3, §22.14

---

## Summary

`BondEscrow::release(principal_idx, amount, tag)` credits the full escrowed amount back to the principal's stake and zeroes the tag. Called during `finalise_expired_slashes` (DSL-031) to return the reporter bond after challenge window passes without winning appeal.

---

## Specification

```rust
fn release(&mut self, principal_idx: u32, amount: u64, tag: BondTag) -> Result<(), BondError>;
```

---

## Acceptance Criteria

- [ ] Escrow zeroed post-release.
- [ ] Stake credited by full amount.
- [ ] Release on empty tag → TagNotFound.
