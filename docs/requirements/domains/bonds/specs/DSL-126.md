# DSL-126: BOND_AWARD_TO_WINNER_BPS = 5_000 (50/50 Split)

> **Authoritative:** [DSL-126](../NORMATIVE.md#DSL-126) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §2.6, §22.14

---

## Summary

`BOND_AWARD_TO_WINNER_BPS = 5_000` (50% of `BPS_DENOMINATOR`). On every adjudication (sustained OR rejected), the forfeited bond is split 50% to the winner, 50% burned.

On sustained: `appellant_award = forfeited * 5_000 / 10_000`; `burn = forfeited - appellant_award`.
On rejected: `reporter_award = forfeited * 5_000 / 10_000`; `burn = forfeited - reporter_award`.

## Acceptance Criteria

- [ ] Constant value == 5_000.
- [ ] Sustained: appellant_award == forfeited × 5_000 / BPS_DENOMINATOR.
- [ ] Rejected: reporter_award == forfeited × 5_000 / BPS_DENOMINATOR.
- [ ] Rounding: integer division; forfeited=3 → award=1, burn=2.
