# DSL-121: BondEscrow::lock InsufficientBalance

> **Authoritative:** [DSL-121](../NORMATIVE.md#DSL-121) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §12.3, §22.14

---

## Summary

`BondEscrow::lock(principal_idx, amount, tag)` MUST return `Err(BondError::InsufficientBalance { have, need })` when the principal has less than `amount` available stake. A successful lock makes `escrowed(principal_idx, tag)` return `amount`.

---

## Specification

```rust
pub trait BondEscrow {
    fn lock(&mut self, principal_idx: u32, amount: u64, tag: BondTag) -> Result<(), BondError>;
    fn escrowed(&self, principal_idx: u32, tag: BondTag) -> u64;
}
```

---

## Acceptance Criteria

- [ ] Sufficient stake → Ok, `escrowed` reflects lock.
- [ ] Insufficient stake → `InsufficientBalance { have, need }`.
- [ ] Tag uniqueness: same (principal, tag) double-lock → `DoubleLock`.
