# DSL-124: REPORTER_BOND_MOJOS = MIN_EFFECTIVE_BALANCE / 64

> **Authoritative:** [DSL-124](../NORMATIVE.md#DSL-124) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §2.6, §22.14

---

## Summary

`REPORTER_BOND_MOJOS = MIN_EFFECTIVE_BALANCE / 64`. This is the bond a reporter must escrow via `BondEscrow::lock` on `submit_evidence`. Small enough to enable low-stake validators to report; large enough to deter grief-reporting.

---

## Specification

```rust
pub const REPORTER_BOND_MOJOS: u64 = MIN_EFFECTIVE_BALANCE / 64;
```

`MIN_EFFECTIVE_BALANCE` re-exported from `dig_consensus::MIN_VALIDATOR_COLLATERAL`.
