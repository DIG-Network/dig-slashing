# DSL-122: BondEscrow::forfeit Returns Mojos

> **Authoritative:** [DSL-122](../NORMATIVE.md#DSL-122) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §12.3, §22.14

---

## Summary

`BondEscrow::forfeit(principal_idx, amount, tag)` returns `Ok(forfeited_mojos)` where `forfeited_mojos` equals the escrowed amount for `(principal_idx, tag)`. Zeroes the tag. Subsequent `escrowed(principal_idx, tag)` returns 0.

---

## Specification

```rust
fn forfeit(&mut self, principal_idx: u32, amount: u64, tag: BondTag) -> Result<u64, BondError>;
```

---

## Acceptance Criteria

- [ ] Returns `forfeited_mojos` equal to escrowed amount.
- [ ] Zeroes the tag.
- [ ] `escrowed` returns 0 after forfeit.
- [ ] Forfeit on empty tag → `TagNotFound`.
