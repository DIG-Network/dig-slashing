# DSL-128: SlashingSystem::genesis Initialisation

> **Authoritative:** [DSL-128](../NORMATIVE.md#DSL-128) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §11, §22.15

---

## Summary

`SlashingSystem::genesis(&GenesisParameters)` constructs the at-birth system state:
- empty `SlashingManager::processed`
- empty `PendingSlashBook`
- empty `slashed_in_window`
- `ParticipationTracker` with `initial_validator_count` zero-flagged entries for both previous and current epoch
- zero-vectored `InactivityScoreTracker`

`current_epoch == params.genesis_epoch`. `in_finality_stall(0, 0)` returns `false` (the chain is not stalled at genesis).

---

## Specification

```rust
pub struct GenesisParameters {
    pub genesis_epoch: u64,
    pub initial_validator_count: usize,
    pub network_id: Bytes32,
}

pub struct SlashingSystem {
    pub manager: SlashingManager,
    pub participation: ParticipationTracker,
    pub inactivity: InactivityScoreTracker,
}

impl SlashingSystem {
    pub fn genesis(params: &GenesisParameters) -> Self;
}
```

---

## Acceptance Criteria

- [ ] `manager.processed.is_empty()`.
- [ ] `manager.book.len() == 0`.
- [ ] `manager.current_epoch == params.genesis_epoch`.
- [ ] `participation.previous_epoch.len() == initial_validator_count`.
- [ ] `participation.current_epoch.len() == initial_validator_count`.
- [ ] All participation flags are `ParticipationFlags(0)`.
- [ ] All inactivity scores are 0.
- [ ] `in_finality_stall(0, 0) == false`.
