# DSL-106: Evidence Mempool Policy — Expired Rejected

> **Authoritative:** [DSL-106](../NORMATIVE.md#DSL-106) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.3, §2.7, §22.12

---

## Summary

`enforce_slashing_evidence_mempool_policy` MUST reject evidence whose `epoch + SLASH_LOOKBACK_EPOCHS < current_epoch` via `SlashingRemarkError::OutsideLookback`. Guards against stale-evidence replay.

---

## Specification

```rust
if current_epoch > SLASH_LOOKBACK_EPOCHS
   && ev.epoch < current_epoch - SLASH_LOOKBACK_EPOCHS {
    return Err(SlashingRemarkError::OutsideLookback {
        epoch: ev.epoch,
        current_epoch,
    });
}
```

---

## Acceptance Criteria

- [ ] Expired → OutsideLookback.
- [ ] At boundary (`ev.epoch == current_epoch - SLASH_LOOKBACK_EPOCHS`) → ok.
- [ ] Within window → ok.
