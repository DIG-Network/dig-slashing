# DSL-120: Appeal Payload Cap

> **Authoritative:** [DSL-120](../NORMATIVE.md#DSL-120) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.6, §2.6, §22.13

---

## Summary

Appeal whose serialized payload exceeds `MAX_APPEAL_PAYLOAD_BYTES` (131_072) rejected via `SlashingRemarkError::PayloadTooLarge`.

---

## Specification

```rust
let len = serde_json::to_vec(ap)?.len();
if len > MAX_APPEAL_PAYLOAD_BYTES {
    return Err(SlashingRemarkError::PayloadTooLarge { actual: len, limit: MAX_APPEAL_PAYLOAD_BYTES });
}
```

## Acceptance Criteria
