# DSL-108: Evidence Block Cap

> **Authoritative:** [DSL-108](../NORMATIVE.md#DSL-108) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.3, §2.8, §22.12

---

## Summary

`enforce_block_level_slashing_caps` MUST reject a block whose evidence count exceeds `MAX_SLASH_PROPOSALS_PER_BLOCK` (64) via `SlashingRemarkError::BlockCapExceeded { actual, limit }`.

---

## Specification

```rust
if evidences.len() > MAX_SLASH_PROPOSALS_PER_BLOCK {
    return Err(SlashingRemarkError::BlockCapExceeded {
        actual: evidences.len(),
        limit: MAX_SLASH_PROPOSALS_PER_BLOCK,
    });
}
```

---

## Acceptance Criteria

- [ ] 65 evidences → BlockCapExceeded.
- [ ] 64 evidences → ok (at boundary).
- [ ] 0 evidences → ok.
