# DSL-117: Appeal Mempool AppealVariantMismatch

> **Authoritative:** [DSL-117](../NORMATIVE.md#DSL-117) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.6, §22.13

---

## Summary

Mempool rejects appeal whose payload variant disagrees with the pending evidence variant via `SlashingRemarkError::AppealVariantMismatch`.

---

## Specification

```rust
match (&appeal.payload, &pending.evidence.payload) {
    (SlashAppealPayload::Proposer(_),     SlashingEvidencePayload::Proposer(_))     => (),
    (SlashAppealPayload::Attester(_),     SlashingEvidencePayload::Attester(_))     => (),
    (SlashAppealPayload::InvalidBlock(_), SlashingEvidencePayload::InvalidBlock(_)) => (),
    _ => return Err(SlashingRemarkError::AppealVariantMismatch),
}
```
