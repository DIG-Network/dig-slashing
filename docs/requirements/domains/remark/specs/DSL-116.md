# DSL-116: Appeal Mempool AppealForFinalisedSlash

> **Authoritative:** [DSL-116](../NORMATIVE.md#DSL-116) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.6, §22.13

---

## Summary

Mempool rejects appeal referencing a `PendingSlash` whose status is `Finalised` OR `Reverted` via `SlashingRemarkError::AppealForFinalisedSlash(hex_hash)`.

---

## Specification

```rust
match pending.status {
    PendingSlashStatus::Finalised { .. } | PendingSlashStatus::Reverted { .. } => {
        return Err(SlashingRemarkError::AppealForFinalisedSlash(hex::encode(...)));
    }
    _ => {}
}
```

---

## Acceptance Criteria

- [ ] Finalised → AppealForFinalisedSlash.
- [ ] Reverted → AppealForFinalisedSlash.
- [ ] Accepted → ok.
- [ ] ChallengeOpen → ok.
