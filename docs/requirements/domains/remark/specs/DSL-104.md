# DSL-104: Evidence Admission Matching Coin

> **Authoritative:** [DSL-104](../NORMATIVE.md#DSL-104) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.2, §22.12

---

## Summary

`enforce_slashing_evidence_remark_admission(bundle, conditions)` MUST return `Ok(())` when the spent coin's `puzzle_hash` equals `slashing_evidence_remark_puzzle_hash_v1(&evidence)` for every parsed evidence in the bundle's conditions.

---

## Specification

```rust
for spend in bundle.coin_spends.iter() {
    let conds = conditions.get(&spend.coin.coin_id()).cloned().unwrap_or_default();
    for ev in parse_slashing_evidence_from_conditions(&hashmap!{spend.coin.coin_id() => conds}) {
        let expected = slashing_evidence_remark_puzzle_hash_v1(&ev)?;
        if spend.coin.puzzle_hash != expected {
            return Err(AdmissionPuzzleHashMismatch { .. });
        }
    }
}
Ok(())
```
