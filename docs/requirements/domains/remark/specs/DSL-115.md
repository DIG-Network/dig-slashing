# DSL-115: Appeal Mempool AppealWindowExpired

> **Authoritative:** [DSL-115](../NORMATIVE.md#DSL-115) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.6, §22.13

---

## Summary

Mempool rejects appeal filed past `submitted_at_epoch + SLASH_APPEAL_WINDOW_EPOCHS` via `SlashingRemarkError::AppealWindowExpired(hex_hash)`.

---

## Specification

```rust
let pending = ... ;
if appeal.filed_epoch > pending.submitted_at_epoch + SLASH_APPEAL_WINDOW_EPOCHS {
    return Err(SlashingRemarkError::AppealWindowExpired(hex::encode(...)));
}
```
