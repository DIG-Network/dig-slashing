> **Authoritative:** [DSL-110](../NORMATIVE.md#DSL-110) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.4, §22.13

## Summary

`encode_slash_appeal_remark_payload_v1` produces `SLASH_APPEAL_REMARK_MAGIC_V1 || serde_json::to_vec(appeal)`. `parse_slash_appeals_from_conditions` inverts. Roundtrip byte-exact.

---

## Specification

```rust
pub fn encode_slash_appeal_remark_payload_v1(ap: &SlashAppeal) -> Result<Vec<u8>, _> {
    let mut out = SLASH_APPEAL_REMARK_MAGIC_V1.to_vec();
    out.extend_from_slice(&serde_json::to_vec(ap)?);
    Ok(out)
}
```
