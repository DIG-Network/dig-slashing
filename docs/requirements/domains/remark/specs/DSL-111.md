## Summary

`slash_appeal_remark_puzzle_reveal_v1` yields a CLVM program emitting exactly one `Condition::Remark` parseable back to the original appeal. `slash_appeal_remark_puzzle_hash_v1` derives via `clvm_utils::tree_hash`.

