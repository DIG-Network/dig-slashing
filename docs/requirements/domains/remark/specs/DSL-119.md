# DSL-119: Appeal Block Cap

> **Authoritative:** [DSL-119](../NORMATIVE.md#DSL-119) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §16.6, §2.8, §22.13

---

## Summary

`enforce_block_level_appeal_caps` MUST reject a block whose appeal count exceeds `MAX_APPEALS_PER_BLOCK` (64) via `SlashingRemarkError::BlockCapExceeded`.

---

## Specification

```rust
if appeals.len() > MAX_APPEALS_PER_BLOCK {
    return Err(SlashingRemarkError::BlockCapExceeded { actual: appeals.len(), limit: MAX_APPEALS_PER_BLOCK });
}
```

## Acceptance Criteria

- [ ] 64 → ok.
