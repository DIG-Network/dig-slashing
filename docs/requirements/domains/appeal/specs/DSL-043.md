# DSL-043: Attester Appeal EmptyIntersection Sustained

> **Authoritative:** [DSL-043](../NORMATIVE.md#DSL-043) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.3, §22.5

---

## Summary

Sustains when `attestation_a.attesting_indices ∩ attestation_b.attesting_indices` is empty. No validator participated in both attestations; the evidence is non-actionable.

---

## Specification

```rust
if evidence.slashable_indices().is_empty() {
    return AppealVerdict::Sustained { reason: AppealSustainReason::EmptyIntersection };
}
```

---

## Acceptance Criteria

- [ ] Disjoint index sets → Sustained.
- [ ] Non-empty intersection → Rejected.
