# DSL-056: submit_appeal WindowExpired

> **Authoritative:** [DSL-056](../NORMATIVE.md#DSL-056) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §22.7

---

## Summary

Rejects when `appeal.filed_epoch > pending.window_expires_at_epoch`. Appeal window is `SLASH_APPEAL_WINDOW_EPOCHS` (8). Bond NOT locked.

---

## Specification

```rust
if appeal.filed_epoch > pending.window_expires_at_epoch {
    return Err(AppealError::WindowExpired {
        submitted_at: pending.submitted_at_epoch,
        window: SLASH_APPEAL_WINDOW_EPOCHS,
        current: appeal.filed_epoch,
    });
}
```

---

## Acceptance Criteria

- [ ] `filed_epoch == window_expires_at_epoch` → accepted (boundary).
- [ ] `filed_epoch == window_expires_at_epoch + 1` → WindowExpired.
- [ ] Bond NOT locked on rejection.
- [ ] Error fields populated.
