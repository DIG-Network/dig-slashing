# DSL-060: submit_appeal SlashAlreadyReverted

> **Authoritative:** [DSL-060](../NORMATIVE.md#DSL-060) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §22.7

---

## Summary

Rejects when `pending.status` is `Reverted { .. }`. Once an appeal succeeds, the slash is non-actionable; further appeals are not accepted.

---

## Specification

```rust
if matches!(pending.status, PendingSlashStatus::Reverted { .. }) {
    return Err(AppealError::SlashAlreadyReverted);
}
```

---

## Acceptance Criteria

- [ ] Post-sustained revert → appeal rejected.
- [ ] Accepted or ChallengeOpen status → accepted.
- [ ] Bond NOT locked.
