# DSL-065: Adjudicate Sustained Reverts Collateral

> **Authoritative:** [DSL-065](../NORMATIVE.md#DSL-065) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §22.8

---

## Summary

When a `CollateralSlasher` is supplied, sustained-appeal adjudication MUST call `CollateralSlasher::credit(idx, collateral_slashed)` per reverted validator. No-op when collateral is `None`.

---

## Specification

```rust
for slash in &pending.base_slash_per_validator {
    if slash.collateral_slashed > 0 {
        if let Some(coll) = collateral.as_mut() {
            coll.credit(slash.validator_index, slash.collateral_slashed)?;
        }
    }
}
```

---

## Acceptance Criteria

- [ ] Collateral supplied → credit called per validator with `collateral_slashed`.
- [ ] Collateral absent → no calls.
- [ ] Zero collateral_slashed → skipped.
