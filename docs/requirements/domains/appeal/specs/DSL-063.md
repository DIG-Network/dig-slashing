# DSL-063: submit_appeal PayloadTooLarge

> **Authoritative:** [DSL-063](../NORMATIVE.md#DSL-063) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §2.6, §22.7

---

## Summary

Rejects when `serialized(appeal).len() > MAX_APPEAL_PAYLOAD_BYTES` (131_072). Bound memory + bound invalid-block witness storage cost. Bond NOT locked.

---

## Specification

```rust
let serialized_len = serde_json::to_vec(&appeal)?.len();
if serialized_len > MAX_APPEAL_PAYLOAD_BYTES {
    return Err(AppealError::PayloadTooLarge {
        actual: serialized_len,
        limit: MAX_APPEAL_PAYLOAD_BYTES,
    });
}
```
