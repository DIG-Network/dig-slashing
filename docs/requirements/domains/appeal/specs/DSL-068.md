# DSL-068: Adjudicate Sustained Reporter-Bond 50/50 Split

> **Authoritative:** [DSL-068](../NORMATIVE.md#DSL-068) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §2.6, §22.8

---

## Summary

Sustained-appeal adjudication MUST forfeit the reporter bond and split it 50/50:
- `appellant_award = forfeited * BOND_AWARD_TO_WINNER_BPS / BPS_DENOMINATOR` (= 50%)
- `burn = forfeited - appellant_award` (= 50%)

`reward_payout.pay(appellant_ph, appellant_award)`.

---

## Specification

```rust
let forfeited = bond_escrow.forfeit(
    reporter_idx,
    REPORTER_BOND_MOJOS,
    BondTag::Reporter(pending.evidence_hash),
)?;
let appellant_award = forfeited * BOND_AWARD_TO_WINNER_BPS as u64 / BPS_DENOMINATOR;
let burn = forfeited - appellant_award;
reward_payout.pay(appeal.appellant_puzzle_hash, appellant_award);
```

---

## Acceptance Criteria

- [ ] `BondEscrow::forfeit` called on reporter.
- [ ] `appellant_award == forfeited * 5_000 / 10_000`.
- [ ] `burn == forfeited - appellant_award`.
- [ ] `RewardPayout::pay` called on `appellant_puzzle_hash` with `appellant_award`.
- [ ] Rounding: integer division; forfeited=1 → award=0, burn=1; forfeited=2 → award=1, burn=1.
