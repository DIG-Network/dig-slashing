# DSL-061: submit_appeal SlashAlreadyFinalised

> **Authoritative:** [DSL-061](../NORMATIVE.md#DSL-061) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §22.7

---

## Summary

Rejects when `pending.status` is `Finalised { .. }`. Window closed, correlation penalty applied, exit lock started — no appeals accepted.

---

## Specification

```rust
if matches!(pending.status, PendingSlashStatus::Finalised { .. }) {
    return Err(AppealError::SlashAlreadyFinalised);
}
```

---

## Acceptance Criteria

- [ ] Finalised status → appeal rejected.
- [ ] Accepted or ChallengeOpen status → accepted.
- [ ] Bond NOT locked on rejection.
