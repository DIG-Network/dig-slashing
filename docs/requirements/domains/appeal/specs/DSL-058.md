# DSL-058: submit_appeal DuplicateAppeal

> **Authoritative:** [DSL-058](../NORMATIVE.md#DSL-058) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §22.7

---

## Summary

Rejects when an appeal's byte-equal payload already exists in `pending.appeal_history`. Prevents spamming the adjudicator with identical rejected appeals. Bond NOT locked.

---

## Specification

```rust
let appeal_hash = appeal.hash();
if pending.appeal_history.iter().any(|a| a.appeal_hash == appeal_hash) {
    return Err(AppealError::DuplicateAppeal);
}
```

---

## Acceptance Criteria

- [ ] Byte-equal prior attempt → DuplicateAppeal.
- [ ] Near-duplicate (different witness bytes) → accepted.
- [ ] Different ground → accepted.
- [ ] Bond NOT locked on rejection.
