# DSL-067: Adjudicate Sustained Clawback wb + prop Rewards

> **Authoritative:** [DSL-067](../NORMATIVE.md#DSL-067) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §12.2, §22.8

---

## Summary

Sustained-appeal adjudication MUST claw back the optimistic `whistleblower_reward` and `proposer_reward` paid at evidence admission. Uses `RewardClawback::claw_back`. Any shortfall (principal already withdrew) is absorbed from the forfeited reporter bond (DSL-073).

---

## Specification

```rust
let wb_amount = /* recomputed from total_eff / 512 */;
let prop_amount = wb_amount / PROPOSER_REWARD_QUOTIENT;
let got_wb = reward_clawback.claw_back(pending.evidence.reporter_puzzle_hash, wb_amount);
let got_prop = reward_clawback.claw_back(proposer_ph, prop_amount);
let shortfall = (wb_amount + prop_amount) - (got_wb + got_prop);
```

---

## Acceptance Criteria

- [ ] Clawback called on reporter puzzle hash with `wb_amount`.
- [ ] Clawback called on proposer puzzle hash with `prop_amount`.
- [ ] Shortfall is tracked into `AppealAdjudicationResult::clawback_shortfall`.
- [ ] Full-clawback case: `shortfall == 0`.
