## Summary

Sustains when `header_a.slot != header_b.slot`. Equivocation requires same-slot; different slots prove the evidence doesn't meet the slashable predicate.

---

## Specification

```rust
if evidence.signed_header_a.message.slot != evidence.signed_header_b.message.slot {
    return AppealVerdict::Sustained { reason: AppealSustainReason::SlotMismatch };
}
```

---

## Acceptance Criteria

- [ ] Different slots → Sustained.
- [ ] Same slot → Rejected.
- [ ] Adjacent slots (off-by-one) → Sustained.
