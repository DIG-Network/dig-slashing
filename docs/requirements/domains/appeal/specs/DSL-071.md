# DSL-071: Adjudicate Rejected Appellant-Bond 50/50 Split

> **Authoritative:** [DSL-071](../NORMATIVE.md#DSL-071) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §22.8

---

## Summary

On `AppealVerdict::Rejected`, the adjudicator MUST forfeit the appellant bond and split 50/50:
- `reporter_award = forfeited * BOND_AWARD_TO_WINNER_BPS / BPS_DENOMINATOR` (50%)
- `burn = forfeited - reporter_award` (50%)

`reward_payout.pay(reporter_ph, reporter_award)`.

---

## Specification

```rust
let forfeited = bond_escrow.forfeit(
    appeal.appellant_index,
    APPELLANT_BOND_MOJOS,
    BondTag::Appellant(appeal.hash()),
)?;
let reporter_award = forfeited * BOND_AWARD_TO_WINNER_BPS as u64 / BPS_DENOMINATOR;
let burn = forfeited - reporter_award;
reward_payout.pay(pending.evidence.reporter_puzzle_hash, reporter_award);
```
