# DSL-042: Attester Appeal NotSlashableByPredicate Sustained

> **Authoritative:** [DSL-042](../NORMATIVE.md#DSL-042) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.3, §22.5

---

## Summary

Sustains when NEITHER the double-vote predicate NOR the surround-vote predicate holds. Re-evaluates the two predicates from DSL-014/DSL-015 against the stored evidence.

---

## Specification

```rust
let dv = a.data.target.epoch == b.data.target.epoch && a.data != b.data;
let sv = (a.data.source.epoch < b.data.source.epoch && a.data.target.epoch > b.data.target.epoch)
      || (b.data.source.epoch < a.data.source.epoch && b.data.target.epoch > a.data.target.epoch);
if !(dv || sv) {
    return AppealVerdict::Sustained { reason: AppealSustainReason::NotSlashableByPredicate };
}
```

---

## Acceptance Criteria

- [ ] Neither predicate holds → Sustained.
- [ ] Double-vote holds → Rejected.
- [ ] Surround-vote holds → Rejected.
- [ ] Non-overlapping epoch windows → Sustained.
