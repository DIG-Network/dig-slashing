# DSL-041: Attester Appeal AttestationsIdentical Sustained

> **Authoritative:** [DSL-041](../NORMATIVE.md#DSL-041) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.3, §22.5

---

## Summary

Sustains when `attestation_a == attestation_b` byte-wise. Identical attestations cannot be a slashable double-vote or surround-vote; any such `PendingSlash` is a buggy verifier admission.

---

## Specification

```rust
if evidence.attestation_a == evidence.attestation_b {
    return AppealVerdict::Sustained { reason: AppealSustainReason::AttestationsIdentical };
}
```

---

## Acceptance Criteria

- [ ] Byte-equal attestations → Sustained.
- [ ] Distinct attestations → Rejected.
- [ ] Determinism.
