# DSL-066: Adjudicate Sustained Restores Validator Status

> **Authoritative:** [DSL-066](../NORMATIVE.md#DSL-066) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §22.8

---

## Summary

Sustained-appeal adjudication MUST call `ValidatorEntry::restore_status()` for every reverted validator. Clears the `Slashed` state set by `submit_evidence` (DSL-022).

---

## Specification

```rust
for slash in &pending.base_slash_per_validator {
    if indices_to_revert.contains(&slash.validator_index) {
        validator_set.get_mut(slash.validator_index).unwrap().restore_status();
    }
}
```

---

## Acceptance Criteria

- [ ] `restore_status` called for every reverted validator.
- [ ] Validator status transitions Slashed → Active.
- [ ] `is_slashed()` returns false post-call.
- [ ] `restore_status` returns true when status changed.
