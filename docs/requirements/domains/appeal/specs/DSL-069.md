# DSL-069: Adjudicate Sustained Reporter Penalty

> **Authoritative:** [DSL-069](../NORMATIVE.md#DSL-069) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.5, §22.8

---

## Summary

Sustained appeal means the reporter filed false/buggy evidence. The adjudicator MUST slash the reporter using the `InvalidBlock` base formula:
```
reporter_penalty = max(eff_bal * INVALID_BLOCK_BASE_BPS / 10_000, eff_bal / MIN_SLASHING_PENALTY_QUOTIENT)
```
Recorded into `slashed_in_window` for correlation amplification at finalisation.

---

## Specification

```rust
let reporter_idx = pending.evidence.reporter_validator_index;
let eff_bal = effective_balances.get(reporter_idx);
let reporter_penalty = std::cmp::max(
    eff_bal * INVALID_BLOCK_BASE_BPS as u64 / BPS_DENOMINATOR,
    eff_bal / MIN_SLASHING_PENALTY_QUOTIENT,
);
validator_set.get_mut(reporter_idx).unwrap().slash_absolute(reporter_penalty, current_epoch);
slashed_in_window.insert((current_epoch, reporter_idx), eff_bal);
```
