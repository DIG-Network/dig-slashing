## Summary

Sustains when `validator_view.get(proposer_index)?.is_active_at_epoch(header.epoch)` returns `false`. Validators outside their active window cannot be proposers of the claimed block.

Witness carries an `ActivationRange { activation_epoch, exit_epoch }` for explorer display, but the authoritative predicate is `ValidatorView`.

---

## Specification

```rust
let entry = validator_view.get(proposer_index).ok_or(AppealError::VerifierError("no validator".into()))?;
if !entry.is_active_at_epoch(header.epoch) {
    return AppealVerdict::Sustained { reason: AppealSustainReason::ValidatorNotActiveAtEpoch };
}
```

---

## Acceptance Criteria

- [ ] `header.epoch < activation_epoch` → Sustained.
- [ ] `header.epoch >= exit_epoch` → Sustained.
- [ ] Active at epoch → Rejected.
- [ ] Boundary epochs: `activation_epoch` active, `exit_epoch - 1` active.
