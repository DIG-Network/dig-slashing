# DSL-059: submit_appeal TooManyAttempts

> **Authoritative:** [DSL-059](../NORMATIVE.md#DSL-059) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §6.1, §2.6, §22.7

---

## Summary

Rejects when `pending.appeal_history.len() >= MAX_APPEAL_ATTEMPTS_PER_SLASH` (4). Caps distinct appeals per slash to bound adjudication cost. Bond NOT locked.

---

## Specification

```rust
if pending.appeal_history.len() >= MAX_APPEAL_ATTEMPTS_PER_SLASH {
    return Err(AppealError::TooManyAttempts {
        count: pending.appeal_history.len(),
        limit: MAX_APPEAL_ATTEMPTS_PER_SLASH,
    });
}
```

---

## Acceptance Criteria

- [ ] 4 rejected prior attempts → 5th submission rejected.
- [ ] 3 prior attempts → 4th accepted.
- [ ] Bond NOT locked.
- [ ] Sustained attempts do not count (they transition to Reverted).
