# DSL-091: InactivityScoreTracker Penalty No-Stall Empty

> **Authoritative:** [DSL-091](../NORMATIVE.md#DSL-091) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §9.3, §22.10

---

## Summary

`epoch_penalties(effective_balances, in_finality_stall)` returns an empty `Vec<(u32, u64)>` when `in_finality_stall == false`. Inactivity penalties only debit during a stall; outside the stall, scores decay (DSL-090) but never charge validators.

---

## Specification

```rust
pub fn epoch_penalties(
    &self,
    effective_balances: &dyn EffectiveBalanceView,
    in_finality_stall: bool,
) -> Vec<(u32, u64)> {
    if !in_finality_stall {
        return Vec::new();
    }
    // ... DSL-092 formula
}
```
