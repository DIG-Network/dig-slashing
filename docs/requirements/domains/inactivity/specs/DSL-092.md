# DSL-092: InactivityScoreTracker Penalty Formula

> **Authoritative:** [DSL-092](../NORMATIVE.md#DSL-092) | **Verification:** [VERIFICATION.md](../VERIFICATION.md) | **Tracking:** [TRACKING.yaml](../TRACKING.yaml) | **SPEC:** §9.3, §2.4, §22.10

---

## Summary

During a stall, per-validator inactivity penalty:
```
penalty = effective_balance * inactivity_score / INACTIVITY_PENALTY_QUOTIENT
                                                 (= 16_777_216)
```

Zero-score validators are filtered out of the result.

---

## Specification

```rust
let mut out = Vec::new();
for (i, &score) in self.scores.iter().enumerate() {
    if score == 0 { continue; }
    let eff_bal = effective_balances.get(i as u32);
    let penalty = (eff_bal as u128 * score as u128 / INACTIVITY_PENALTY_QUOTIENT as u128) as u64;
    if penalty > 0 {
        out.push((i as u32, penalty));
    }
}
```

---

## Acceptance Criteria

- [ ] Formula: `eff_bal * score / 16_777_216`.
- [ ] Zero-score validators omitted.
- [ ] u128 intermediate to avoid u64 overflow for `eff_bal * score`.
- [ ] Only applied when `in_finality_stall == true`.
