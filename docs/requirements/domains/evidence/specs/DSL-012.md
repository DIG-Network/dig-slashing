## Summary

A validator cannot be both reporter and accused in the same evidence. `verify_evidence` rejects with `SlashingError::ReporterIsAccused(reporter_index)` when `evidence.reporter_validator_index ∈ evidence.slashable_validators()`. Prevents a validator self-slashing to collect the whistleblower reward.

---

## Specification

```rust
let slashable = evidence.slashable_validators();
if slashable.contains(&evidence.reporter_validator_index) {
    return Err(SlashingError::ReporterIsAccused(evidence.reporter_validator_index));
}
```

---

## Acceptance Criteria

- [ ] Reporter outside the slashable set → passes through to next check.
- [ ] Reporter in the slashable set → `ReporterIsAccused(index)`.
- [ ] Error carries the offending index.
- [ ] No state mutation on rejection.
