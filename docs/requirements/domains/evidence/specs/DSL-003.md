## Summary

`Checkpoint` is the FFG vote `(epoch, root)` pair used in `AttestationData.source` and `.target`. Required traits: `Copy`, `Eq`, `Hash`, `PartialEq`, `Serialize`, `Deserialize`. Serde round-trip must preserve byte-exact equality.

---

## Specification

```rust
#[derive(Debug, Clone, Copy, Serialize, Deserialize, PartialEq, Eq, Hash)]
pub struct Checkpoint {
    pub epoch: u64,
    pub root: Bytes32,
}
```

---

## Acceptance Criteria

- [ ] Derives `Debug, Clone, Copy, Serialize, Deserialize, PartialEq, Eq, Hash`.
- [ ] `bincode` round-trip preserves equality.
- [ ] `serde_json` round-trip preserves equality.
- [ ] Two `Checkpoint` values with identical fields hash to the same `u64` via `Hash`.
