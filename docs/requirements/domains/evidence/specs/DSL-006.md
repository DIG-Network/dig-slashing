## Summary

`IndexedAttestation::verify_signature` reuses `chia_bls::aggregate_verify` to check the aggregate BLS signature across every committee member's public key against `signing_root(network_id)`. A single valid aggregate → `Ok(())`. Any corruption (signature mutation, substituted pubkey, tampered message) → `Err(SlashingError::BlsVerifyFailed)`.

---

## Specification

```rust
impl IndexedAttestation {
    pub fn verify_signature(
        &self,
        pks: &dyn PublicKeyLookup,
        network_id: &Bytes32,
    ) -> Result<(), SlashingError> {
        let signing_root = self.data.signing_root(network_id);
        let pubkeys: Vec<PublicKey> = self
            .attesting_indices
            .iter()
            .filter_map(|i| pks.pubkey_of(*i).cloned())
            .collect();
        let sig_bytes: [u8; BLS_SIGNATURE_SIZE] = self
            .signature
            .as_slice()
            .try_into()
            .map_err(|_| SlashingError::BlsVerifyFailed)?;
        let sig = Signature::from_bytes(&sig_bytes).map_err(|_| SlashingError::BlsVerifyFailed)?;
        let msgs: Vec<&[u8]> = std::iter::repeat(signing_root.as_ref())
            .take(pubkeys.len())
            .collect();
        if !chia_bls::aggregate_verify(&sig, pubkeys.iter().zip(msgs.iter().copied())) {
            return Err(SlashingError::BlsVerifyFailed);
        }
        Ok(())
    }
}
```

---

## Acceptance Criteria

- [ ] Valid aggregate over committee pubkeys → `Ok(())`.
- [ ] One-bit mutation of `signature` → `BlsVerifyFailed`.
- [ ] Substituted pubkey → `BlsVerifyFailed`.
- [ ] Mutated message (signing_root shifted) → `BlsVerifyFailed`.
- [ ] Missing pubkey (index not in lookup) → `BlsVerifyFailed` (aggregate fails).
- [ ] Uses `chia_bls::aggregate_verify` (no custom BLS).
