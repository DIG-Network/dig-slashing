## Summary

`SignedBlockHeader` wraps a `dig_block::L2BlockHeader` with a 96-byte BLS signature. Used in both `ProposerSlashing` (two `SignedBlockHeader`s) and `InvalidBlockProof` (one). Serde must roundtrip byte-exactly for REMARK and `PendingSlash` wire forms.

---

## Specification

```rust
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq, Eq)]
pub struct SignedBlockHeader {
    pub message: L2BlockHeader,
    #[serde(with = "serde_bytes")]
    pub signature: Vec<u8>,
}
```

---

## Acceptance Criteria

- [ ] `message` is `dig_block::L2BlockHeader` (not redefined here).
- [ ] `signature` uses `serde_bytes` (raw bytes, not array).
- [ ] `bincode` round-trip preserves byte-exact equality.
- [ ] `serde_json` round-trip preserves equality.
- [ ] Signature length is enforced to 96 bytes downstream by verifiers, not in the type itself.
