## Summary

Even when the double-vote or surround-vote predicate holds, the evidence is only actionable if at least one validator participated in BOTH attestations. `verify_attester_slashing` MUST return `SlashingError::EmptySlashableIntersection` when `AttesterSlashing::slashable_indices()` is empty.

---

## Specification

```rust
let slashable = payload.slashable_indices();
if slashable.is_empty() {
    return Err(SlashingError::EmptySlashableIntersection);
}
```

---

## Acceptance Criteria

- [ ] Disjoint attesting_indices → `EmptySlashableIntersection`.
- [ ] Non-empty intersection → passes through to next check.
- [ ] Check runs AFTER predicate validation (DSL-014/015) and structure validation (DSL-005).
- [ ] Check runs BEFORE signature verification (avoid wasted BLS work).
