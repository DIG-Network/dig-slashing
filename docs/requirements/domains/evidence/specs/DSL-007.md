## Summary

`AttesterSlashing::slashable_indices()` computes the set of validators who participated in BOTH offending attestations. Deterministic, sorted (strictly ascending, no duplicates), O(n+m) via two-pointer walk assuming both input index lists are already ascending. Result feeds into the per-validator slash loop in `SlashingManager::submit_evidence`.

---

## Specification

```rust
impl AttesterSlashing {
    pub fn slashable_indices(&self) -> Vec<u32> {
        let a = &self.attestation_a.attesting_indices;
        let b = &self.attestation_b.attesting_indices;
        let mut out = Vec::new();
        let (mut i, mut j) = (0, 0);
        while i < a.len() && j < b.len() {
            match a[i].cmp(&b[j]) {
                Ordering::Equal   => { out.push(a[i]); i += 1; j += 1; }
                Ordering::Less    => i += 1,
                Ordering::Greater => j += 1,
            }
        }
        out
    }
}
```

---

## Acceptance Criteria

- [ ] Result is strictly ascending.
- [ ] Result contains every value in `a ∩ b`, no duplicates.
- [ ] Disjoint inputs → empty result.
- [ ] Identical inputs → result equals either input.
- [ ] Deterministic (byte-equal output across runs).
